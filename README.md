# 🚀 Sales Performance Analysis & Dashboard
A complete Sales Operations Analyst Intern–ready project built using Excel for analyzing sales performance, regional trends, product profitability, and sales funnel efficiency.

---

## 📌 Project Summary
This project analyzes sales data from multiple regions, products, and customer accounts. Using Excel, we generate KPIs, Pivot Tables, charts, and a complete dashboard that helps in:
- Tracking monthly sales performance
- Comparing revenue across regions
- Analyzing top products
- Understanding the sales funnel
- Identifying trends and business insights

---

## 🛠️ Tools & Technologies Used
- Excel (Pivot Tables, Charts, Dashboard)
- PowerPoint (Project Presentation)
- PDF Report (Documentation)
- Basic Business Analytics Concepts
